Para executar a entrega 04, siga os passos abaixo:

1. Abra o arquivo "Entrega_04_Macrocelulas.m".
2. Execute o arquivo "Entrega_04_Macrocelulas.m". 
	Na Command Window será mostrado o resultado da porcentagem de outage para cada portadora. 
	Será plotado será seis gráfios: três das potências de cada portadora e três da área das cobertura das respectivas frequências.
3. Abra o arquivo "Entrega_04_Macrocelulas_E_Microcelulas.m".
4. Execute o arquivo "Entrega_04_Macrocelulas_E_Microcelulas.m". 
	Na Command Window será mostrado o resultado da porcentagem de outage para cada portadora, com 7 macrocélulas mais 6 microcélulas. 
	Também será plotado seis gráficos aqui, sendo três das potências de cada portadora e três da área das cobertura das respectivas frequências.
