Para executar a entrega 01 da segunda unidade, siga os passos:
	1. Abra o arquivo "Entrega_01.m" no matlab, a versão do programa usada para fazer esse código foi o R2015a.
	2. Rode o programa.
	 - Será plotado um gráfico Pe vs Eb/No para a modulação BPSK e 16-QAM, as modulações podem ser identificadas pela legenda.
	 - Na Command Window será mostrado os valores das variâncias para cada tipo de modulação com o respectivo valor de Eb/No.
	3. Apenas caso deseje visualizar os gráficos das modulações para cada modulação, siga os pontos abaixo. Esses gráficos foram comentados, por serem 30 gráficos e não serem o grande objetivo desta entrega.
	  3.1 Para visualizar os plots da Modulação 16-QAM com diferentes valores de Eb/No, descomente da linha 110 até a linha 114.
	  3.2 Para visualizar os plots da Modulação BPSK com diferentes valores de Eb/No, descomente da linha 174 até a linha 179.
